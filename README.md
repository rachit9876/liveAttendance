## Download & Setup

1. **Download Project**
   - Go to: https://github.com/rachit9876/liveAttendance
   - Click green "Code" button ‚Üí Download ZIP
   - Extract ZIP to your desired location

2. **Install Python 3.10.11**
   - Download from: https://www.python.org/downloads/release/python-31011/
   - Install and check "Add Python to PATH"

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run Application**
   ```bash
   cd liveAttendance-main/backend
   python app.py
   ```

5. **Open Browser**
   - Go to: http://localhost:5000

## Features
- Student Registration
- Face Recognition Attendance
- Live Surveillance with Sleeping Detection
- Attendance Reports
